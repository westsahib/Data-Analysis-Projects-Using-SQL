 Chocolate Sales Analysis using SQL

This project focuses on analyzing chocolate sales data to understand revenue patterns, salesperson performance, product efficiency, and time-based trends.

 Problem Statements & SQL Queries
🟢 Basic

1. How many total transactions are recorded?

SELECT COUNT(*) AS total_transactions
FROM choco;

2. Clean the revenue column by removing symbols and formatting issues

UPDATE choco 
SET amount = REPLACE(REPLACE(amount, '$', ''), ',', '');

3. Convert revenue column into numeric format

ALTER TABLE choco 
MODIFY amount DECIMAL(15,3) NOT NULL;
🟡 Intermediate

4. What is the total revenue generated by each country? Rank them accordingly.

SELECT country,
       SUM(amount) AS total_revenue,
       RANK() OVER (ORDER BY SUM(amount) DESC) AS country_rank
FROM choco
GROUP BY country
ORDER BY total_revenue DESC;

5. How does each salesperson perform in terms of revenue and number of orders?

SELECT salesperson,
       SUM(amount) AS total_sales,
       COUNT(*) AS total_orders,
       RANK() OVER (ORDER BY SUM(amount) DESC) AS salesperson_rank
FROM choco
GROUP BY salesperson;

6. How many sales records exist per country?

SELECT country,
       COUNT(salesperson) AS total_records
FROM choco
GROUP BY country;

7. What are the top 5 products based on total revenue?

SELECT *
FROM (
    SELECT product,
           SUM(amount) AS revenue,
           RANK() OVER (ORDER BY SUM(amount) DESC) AS rank_product
    FROM choco
    GROUP BY product
) t
WHERE rank_product <= 5;

8. Which products generate the highest revenue per box (efficiency)?

SELECT product,
       SUM(amount) / SUM(boxes) AS revenue_per_box
FROM choco
GROUP BY product
ORDER BY revenue_per_box DESC
LIMIT 5;

9. What is the monthly revenue trend?

SELECT DATE_FORMAT(date, '%Y-%m') AS month,
       SUM(amount) AS monthly_revenue
FROM choco
GROUP BY month
ORDER BY month;

10. What percentage of total revenue does each country contribute?

SELECT country,
       SUM(amount) AS revenue,
       ROUND(SUM(amount) / (SELECT SUM(amount) FROM choco) * 100, 3) AS percentage
FROM choco
GROUP BY country
ORDER BY percentage DESC;
🔴 Advanced

11. What is the month-over-month growth in revenue?

WITH monthly AS (
    SELECT DATE_FORMAT(date, '%Y-%m') AS month,
           SUM(amount) AS monthly_revenue
    FROM choco
    GROUP BY month
)
SELECT month,
       monthly_revenue,
       ROUND(
           (monthly_revenue - LAG(monthly_revenue) OVER (ORDER BY month)) /
           LAG(monthly_revenue) OVER (ORDER BY month) * 100, 2
       ) AS growth_percentage
FROM monthly;

12. Which salespersons are performing above average?

SELECT salesperson,
       SUM(amount) AS revenue,
       ROUND(
           SUM(amount) - (
               SELECT AVG(sales)
               FROM (
                   SELECT SUM(amount) AS sales
                   FROM choco
                   GROUP BY salesperson
               ) t
           ), 2
       ) AS performance_score,
       RANK() OVER (ORDER BY performance_score DESC) AS rank_position
FROM choco
GROUP BY salesperson
HAVING performance_score > 0;

13. What is the average revenue generated per salesperson?

SELECT ROUND(AVG(sales), 2) AS avg_sales
FROM (
    SELECT salesperson, SUM(amount) AS sales
    FROM choco
    GROUP BY salesperson
) t;

14. Rank products based on revenue per box (premium products)

SELECT product,
       ROUND(SUM(amount) / SUM(boxes), 3) AS revenue_per_box,
       RANK() OVER (ORDER BY revenue_per_box DESC) AS premium_rank
FROM choco
GROUP BY product;

15. How much do top-performing countries contribute to total revenue?

WITH ranked AS (
    SELECT country,
           SUM(amount) AS revenue,
           RANK() OVER (ORDER BY SUM(amount) DESC) AS rnk
    FROM choco
    GROUP BY country
)
SELECT SUM(revenue) AS top_revenue,
       ROUND(SUM(revenue) / (SELECT SUM(amount) FROM choco) * 100, 2) AS percent_of_total
FROM ranked
WHERE rnk <= 6;

16. What is the revenue generated by each product and salesperson combination?

SELECT product, salesperson, SUM(amount) AS revenue
FROM choco
GROUP BY product, salesperson
ORDER BY revenue DESC;

17. Which salesperson is most efficient in terms of revenue per box?

SELECT salesperson,
       SUM(amount) / SUM(boxes) AS efficiency
FROM choco
GROUP BY salesperson
ORDER BY efficiency DESC;
