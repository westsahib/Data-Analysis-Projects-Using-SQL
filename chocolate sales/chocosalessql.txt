

##  Chocolate Sales Analysis using SQL

---

###  Basic

**1. How many total transactions are recorded?**

```sql
SELECT COUNT(*) AS total_transactions
FROM choco;
```

---

**2. Clean the revenue column**

```sql
UPDATE choco 
SET amount = REPLACE(REPLACE(amount, '$', ''), ',', '');
```

---

**3. Convert revenue column into numeric format**

```sql
ALTER TABLE choco 
MODIFY amount DECIMAL(15,3) NOT NULL;
```

---

### Intermediate

**4. Revenue by country with ranking**

```sql
SELECT country,
       SUM(amount) AS total_revenue,
       RANK() OVER (ORDER BY SUM(amount) DESC) AS country_rank
FROM choco
GROUP BY country
ORDER BY total_revenue DESC;
```

---

**5. Salesperson performance**

```sql
SELECT salesperson,
       SUM(amount) AS total_sales,
       COUNT(*) AS total_orders,
       RANK() OVER (ORDER BY SUM(amount) DESC) AS salesperson_rank
FROM choco
GROUP BY salesperson;
```

---

**6. Records per country**

```sql
SELECT country,
       COUNT(salesperson) AS total_records
FROM choco
GROUP BY country;
```

---

**7. Top 5 products by revenue**

```sql
SELECT *
FROM (
    SELECT product,
           SUM(amount) AS revenue,
           RANK() OVER (ORDER BY SUM(amount) DESC) AS rank_product
    FROM choco
    GROUP BY product
) t
WHERE rank_product <= 5;
```

---

**8. Revenue per box**

```sql
SELECT product,
       SUM(amount) / SUM(boxes) AS revenue_per_box
FROM choco
GROUP BY product
ORDER BY revenue_per_box DESC
LIMIT 5;
```

---

**9. Monthly revenue trend**

```sql
SELECT DATE_FORMAT(date, '%Y-%m') AS month,
       SUM(amount) AS monthly_revenue
FROM choco
GROUP BY month
ORDER BY month;
```

---

**10. Revenue contribution by country**

```sql
SELECT country,
       SUM(amount) AS revenue,
       ROUND(SUM(amount) / (SELECT SUM(amount) FROM choco) * 100, 3) AS percentage
FROM choco
GROUP BY country
ORDER BY percentage DESC;
```

---

###  Advanced

**11. Month-over-month growth**

```sql
WITH monthly AS (
    SELECT DATE_FORMAT(date, '%Y-%m') AS month,
           SUM(amount) AS monthly_revenue
    FROM choco
    GROUP BY month
)
SELECT month,
       monthly_revenue,
       ROUND(
           (monthly_revenue - LAG(monthly_revenue) OVER (ORDER BY month)) /
           LAG(monthly_revenue) OVER (ORDER BY month) * 100, 2
       ) AS growth_percentage
FROM monthly;
```

---

**12. Above-average salespersons**

```sql
SELECT salesperson,
       SUM(amount) AS revenue,
       ROUND(
           SUM(amount) - (
               SELECT AVG(sales)
               FROM (
                   SELECT SUM(amount) AS sales
                   FROM choco
                   GROUP BY salesperson
               ) t
           ), 2
       ) AS performance_score,
       RANK() OVER (ORDER BY performance_score DESC) AS rank_position
FROM choco
GROUP BY salesperson
HAVING performance_score > 0;
```

---

**13. Average sales per salesperson**

```sql
SELECT ROUND(AVG(sales), 2) AS avg_sales
FROM (
    SELECT salesperson, SUM(amount) AS sales
    FROM choco
    GROUP BY salesperson
) t;
```

---

**14. Product premium ranking**

```sql
SELECT product,
       ROUND(SUM(amount) / SUM(boxes), 3) AS revenue_per_box,
       RANK() OVER (ORDER BY revenue_per_box DESC) AS premium_rank
FROM choco
GROUP BY product;
```

---

**15. Contribution of top countries**

```sql
WITH ranked AS (
    SELECT country,
           SUM(amount) AS revenue,
           RANK() OVER (ORDER BY SUM(amount) DESC) AS rnk
    FROM choco
    GROUP BY country
)
SELECT SUM(revenue) AS top_revenue,
       ROUND(SUM(revenue) / (SELECT SUM(amount) FROM choco) * 100, 2) AS percent_of_total
FROM ranked
WHERE rnk <= 6;
```

---

**16. Revenue by product & salesperson**

```sql
SELECT product, salesperson, SUM(amount) AS revenue
FROM choco
GROUP BY product, salesperson
ORDER BY revenue DESC;
```

---

**17. Salesperson efficiency**

```sql
SELECT salesperson,
       SUM(amount) / SUM(boxes) AS efficiency
FROM choco
GROUP BY salesperson
ORDER BY efficiency DESC;
```
