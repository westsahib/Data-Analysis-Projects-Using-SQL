# Data-Analysis-Projects-Using-SQL

This repository is a collection of SQL-based data analysis projects where I explore structured datasets and try to extract meaningful insights from them.

The idea behind these projects is simple — not just to practice SQL syntax, but to understand how data is actually used to answer real business questions.

📁 Projects Included:

🍕 Pizza Sales Analysis

In this project, I analyzed a pizza sales dataset to understand:

Revenue trends
Most popular pizzas and sizes
Order distribution across time
Top-performing products

I used joins, aggregations, and window functions to break down the data and uncover patterns.

🍫 Chocolate Sales Analysis

This project focuses more on performance and efficiency:

Revenue by country and salesperson
Product performance (revenue per box)
Monthly revenue trends and growth
Benchmarking sales performance against average

It also includes advanced SQL concepts like CTEs, ranking, and LAG functions.
